Jogo 2d feito com JavaScript!

Como Jogar:

  • Use as setas do teclado para mover o personagem.
  • Pressione a barra de espaço para atirar projéteis nos inimigos.
  • Evite colidir com os inimigos e seus ataques.
  • Colete power-ups para ganhar habilidades especiais e aumentar sua pontuação.
  • Para ganhar, você precisa fazer 120 pontos em um minuto.

Sobre o jogo

Este jogo foi criado utilizando JavaScript puro, sem o uso de frameworks, aproveitando a tag <canvas> para renderizar os gráficos. A estrutura do jogo é baseada em orientação a objetos, o que facilita a organização e a manutenção do código. Cada personagem, inimigo e elemento do jogo é representado como um objeto, permitindo uma interação dinâmica e eficiente entre eles.

Elementos do Jogo:

  • Projéteis: Usados pelo jogador para atacar inimigos.
  • Gears: Partículas geradas durante explosões.
  • Explosões de Fumaça e Fogo: Efeitos visuais para explosões.
  • Escudo: Protege o jogador de ataques inimigos.
  • Fireball:Disparos feitos pelo jogador.

Ambiente:

  • O jogo possui múltiplas camadas de fundo para criar um efeito de paralaxe, adicionando profundidade ao ambiente do jogo.

Sons:

  • O jogo inclui efeitos sonoros para power-ups, explosões, acertos, e outros eventos importantes para melhorar a imersão do jogador.
